
## facenet-pytorch

主干网络支持:
    - Inception_resnet_v1
    - mobilenet_v2

| 主干网络        | 使用分类辅助 | LFW 准确率 |
| --------------- | ------------ | ---------- |
| MobileNetV2     | 否           | 93.06%     |
| MobileNetV2     | 是           | 94.27%     |
| EfficientNet-B0 | 否           | 98.10%     |
| EfficientNet-B0 | 是           | 98.65%     |
